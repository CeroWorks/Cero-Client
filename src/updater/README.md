
# Cero-Updater

## Build It Yourself

You can build the **GoLang Updater**.

**Requirements:**

* **GoLang** ≥ `1.25.3`
* **Node.js** ≥ `v24.11.0`
* **Python** ≥ `3.13`
* **npm** ≥ `v11.6.1`

**Build for your platform (example for Linux):**

```bash
npm run buildUpdater
```

Use Updater interface:

```text
/------- CeroClient Updater -------\
| Choose OS :                      |
| (0) Linux                        |
| (1) Windows                      |
\----------------------------------/
OS : Linux
```

> Replace `Linux` with `Windows` depending on your target OS.
